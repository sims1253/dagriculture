# Define a groots kind

Define a groots kind

## Usage

``` r
groots_kind(
  name,
  input_contract = NULL,
  output_type = NULL,
  param_schema = NULL
)
```

## Arguments

- name:

  The name of the kind.

- input_contract:

  Input contract list.

- output_type:

  Output type string.

- param_schema:

  Parameter schema.
