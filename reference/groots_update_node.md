# Update a node in a groots graph

Update a node in a groots graph

## Usage

``` r
groots_update_node(
  graph,
  node_id,
  label = NULL,
  params = NULL,
  metadata = NULL
)
```

## Arguments

- graph:

  A `groots_graph`.

- node_id:

  Node ID.

- label:

  Node label.

- params:

  Node parameters.

- metadata:

  Node metadata.
