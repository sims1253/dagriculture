# Add an edge to a groots graph

Add an edge to a groots graph

## Usage

``` r
groots_add_edge(graph, from, to, type = "data", id = NULL, metadata = list())
```

## Arguments

- graph:

  A `groots_graph`.

- from:

  Upstream node ID.

- to:

  Downstream node ID.

- type:

  Edge type.

- id:

  Optional Edge ID.

- metadata:

  Edge metadata.
